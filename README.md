### Discord BBS

It's a like an old modem but with discord (or any chat software) as the carrier network (forget that discord sends data over the internet :)).

#### How does it work?
The program uses the PortAudio C library to generate a sine wave that changes frequency to represent a zero or one (frequency shift keying).
If you play the audio in a call, and your friend has a program to decode it (coming) you can transfer data in a really unreliable way :P
